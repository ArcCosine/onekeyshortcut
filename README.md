# One-key shortcut like Presto version Opera.

## Supported key and action

- 1 Move to previous tab.
- 2 Move to next tab.
- j Scroll Down.
- k Scroll Up.

## Future support.

- h Focus address bar
- a Highlight next url
- q Highlight  previous url
- 6 Reset zoom to 100%
- 9 Zoom out
- 0 Zoom in
- f Hit a hint
